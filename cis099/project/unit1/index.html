Unit 1 Project

Instructions

Build a web page that contains the following elements:

  1. A title appropriate to the content of page
  2. a page heading using the largest heading
  3. at least one medium-sized heading (not the largest or smallest)
  4. at least two paragraph tags with content
  5. at least one line break element
  6. some italicized text using html tags
  7. some bold text using html tags
  8. an ordered list with at least 3 items
  9. an unordered list with at least 3 items
  10. at least 2 absolute hyperlinks

To receive credit for this assignment. All of these elements must be present and correct

Content

The web page may contain any content appropriate for a school setting. some examples include:

  • A page about your favorite band with members and albums
  • A recipe page for your favorite meal with ingredients and steps
  • A DIY project with materials and steps

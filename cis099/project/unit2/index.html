Complete the following:

  • Perform the following steps in repl.it:
    1. Add a folder to your root directory called files
    2. Add a folder to the files folder called pages
    3. Add the following files to the pages folder:
      • page1.html
      • page2.html
      • page3.html
  • Your index.html file should contain the following:
    1. a title appropriate to the content of the page
    2. all of the semantic elements covered in this unit
    3. a heading for the page using the largest heading
    4. three hyperlinks to the pages created earlier
    5. a least two paragraph elements
    6. a list structure consisting of at least two nested lists
    7. an ordered list which orders items by lowercase letters (this may be a part of your nested list structure)
    8. an least these character codes which may include: space, &,©,<,>,etc.

In order to receive credit for this assignment, all files and elements must be present, correct, and inside an appropriate sematic element

Content

The content for the this assignment may be anything suitable for a school environment. Feel free to practice your skills by flesh out the page1.html, page2.html, and pages3.html files

Some topics suitable for the nested lists include:

  • a table of contents
  • a list of rankings for unlike things
  • anything requiring a hierarchy
